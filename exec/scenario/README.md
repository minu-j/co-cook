## 3. 냉털
![냉털](search_start.jpg)
- 탭하여 시작 위 부분을 누른다.
- 음성 입력 준비 완료

![냉털](search_end.jpg)
- 돼지고기, 고추장, 두부, 마늘, 양파를 말한다.
- 완료 버튼을 누른다.

![냉털](search.jpg)
- 검색된 메뉴가 나온다.
- 두부, 고추장 등 위에 토글을 켰다 끄면서 바뀌는 화면을 보여준다.
- 즐겨찾기를 등록 및 삭제한다.
